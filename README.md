# Music-Player-with-JDBC
An advanced Java Music Player application integrated with MySQL using JDBC. It supports music library management, playlist creation, and database storage for persistent data handling.


##  Overview
This is an advanced music player application built using Core Java and JDBC. It connects to MySQL to store songs and playlists permanently.

---

##  Features
- Add / View Songs
- Create Playlists
- Add Songs to Playlist
- View Playlist Songs
- MySQL Database Integration
- Automatic Table Creation
- JDBC Connectivity

---

##  Technologies Used
- Java (Core)
- JDBC
- MySQL (XAMPP)
- SQL Queries

---
